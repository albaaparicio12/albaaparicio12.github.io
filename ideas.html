🌍 Lalo - Language Location
✨ on going

Web that shows you how many and which countries you can travel to depending on which langugaes do you speak.

Additional requirements:

  • Skyscanner connection to the country selected.

  • Statistics like with how many people you can communicate, how many countries you can visit, etc.
  • Resources page for each language (map with colored countries, language learning resources, places to visit, spotify music, movies, etc.)
